# kintai-extraction

月末締めの勤怠の自動化 POC

###############################################################

**stage1 構想**

1. 抽出対象の excel、抽出設定.xlsx、kintai.py を同じディレクトリ内に並べる
2. kintai.py を実行し、1 ファイルずつ抽出する

**stage2 構想**

複数ファイルから一気に抽出ができるようにする

**stage3 構想**

抽出対象の excel を、抽出設定.xlsx のパス設定の箇所で好きな階層から選べるようにする
